# Principios SOLID – (Unity)

Aplicación interactiva desarrollada en **Unity** para explorar y comprender los **5 principios SOLID** a través de ejemplos claros y comparativos.

El proyecto permite, para cada principio:
- leer la **descripción del principio**
- analizar un **mal ejemplo** (violación del principio)
- analizar un **buen ejemplo** (aplicación correcta)
- inspeccionar el **código asociado** a cada caso

> **Repositorio principal (código fuente e historial completo):**  
> GitLab: https://gitlab.com/luism.garciac/solid-quark-academy

---

## Qué muestra este proyecto

- Explicación práctica de los **principios SOLID**
- Comparación directa entre:
  - implementación incorrecta
  - implementación correcta
- Código orientado a **comprensión conceptual**, no a producción
- Uso de **Unity como entorno visual e interactivo** para reforzar el aprendizaje

---

## Principios incluidos

- **S** – Single Responsibility Principle  
- **O** – Open / Closed Principle  
- **L** – Liskov Substitution Principle  
- **I** – Interface Segregation Principle  
- **D** – Dependency Inversion Principle  

---

## Cómo ejecutar el proyecto

1. Clonar el repositorio principal desde GitLab
2. Abrir el proyecto en **Unity** (versión LTS recomendada)
3. Abrir la escena principal del proyecto (SolidPrinciples)
4. Ejecutar en el editor (`Play`)

## Estructura del código

Los ejemplos de código asociados a cada principio se encuentran organizados por principio y por tipo de ejemplo (bueno / malo), facilitando su lectura y comparación.

Assets/Sources

Cada principio cuenta con implementaciones independientes para evitar acoplamientos innecesarios entre ejemplos.

---

## Notas sobre el alcance

- Este proyecto **no busca ser un framework ni código de producción**
- El foco está puesto en:
  - claridad conceptual
  - contraste entre buenas y malas prácticas
  - aprendizaje y análisis

Algunas decisiones priorizan simplicidad y didáctica por sobre “best practices” de producto final.

Proyecto educativo desarrollado como material de estudio y referencia.
